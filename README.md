🚀 Repositorio de Sistemas Digitales
Este repositorio contiene los proyectos y exámenes prácticos desarrollados durante el curso de Sistemas Digitales, enfocados en el diseño de hardware avanzado, lógica programable y sistemas embebidos.

📂 Estructura del Proyecto
El repositorio está organizado por hitos académicos, documentando la evolución desde lógica de control hasta sistemas integrados:

🔹 Parcial 1: Control de Acceso (FSM)
Basys3_Parciales: Implementación de un sistema de control para un parqueo.

Utiliza una arquitectura de dos máquinas de estados: una Máquina de Mealy para gestionar el sistema de pago y una Máquina de Moore para el control de la talanquera.

🔹 Parcial 2: Diseño de Videojuegos en FPGA
Parcial2_SD: Desarrollo del juego "UNIS Invaders".

Implementación completa de lógica de video y control de juego ejecutado directamente en la tarjeta Basys 3.

🔹 Parcial 3: Aritmética Digital Avanzada
ALU (Arithmetic Logic Unit): Diseño modular de una unidad aritmética de alto rendimiento.

Incluye implementaciones de sumadores tipo CLA de 4 bits (Carry Look-Ahead) y Prefix Adders.

Funciones adicionales de desplazamiento: Shift Left y Shift Right.

Demostración: Se incluye el archivo ALU_KaterinSagastume_P3.mp4 con la simulación completa de las operaciones.

🔹 Parcial Final: Sistema Transaccional Embebido
Cajero Final: Proyecto integrador que simula un cajero automático.

A diferencia de los módulos anteriores, este sistema fue implementado utilizando la placa NUCLEO-L053R8.

Incluye el código fuente y un video demostrativo del funcionamiento del hardware en tiempo real.

🛠️ Herramientas y Tecnologías
Lenguajes: SystemVerilog / Verilog y C para sistemas embebidos.

**Hardware:**

Basys 3 FPGA (Artix-7) para lógica programable.

STM32 Nucleo-L053R8 para el proyecto final.

Software: Xilinx Vivado (Síntesis e Implementación) y entornos para microcontroladores STM32.
